Workflow Cheatsheet

×

Live Coding Flow

  • 1. Keep this window docked half-screen. Capture snippets via Ctrl+Alt+S (screen) or Ctrl+Alt+W (window).
  • 2. Tracked requirements build automatically — add manual notes any time.
  • 3. When ready, press Ctrl+Alt+L or click “Generate Plan” for a complete response outline.
  • 4. Use the plan as a script for pair-programming or to narrate your approach.

Hotkeys

  • Ctrl+Alt+S – Capture selected screen
  • Ctrl+Alt+W – Capture selected window
  • Ctrl+Alt+L – Generate combined plan
  • Ctrl+Alt+A – Toggle assistance panel
  • Alt+Shift+C – Legacy capture trigger

Tips

Refresh sources if you switch between screens or add a new browser tab capture. The tracker deduplicates notes so you can safely recapture as requirements evolve.

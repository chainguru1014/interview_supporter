# Ethics & Intended Use

This project is designed for **interview practice**, preparation, and transparent assistance.
It must **not** be used to deceive or misrepresent your capabilities in a live assessment.
Examples of allowed use:
- Practicing coding challenges with automated feedback.
- Using a visible assistance panel during a collaborative session, with all parties aware.
- Generating notes and summaries from your own speech and the interviewer's questions.

Prohibited uses include covert prompting, hidden overlays, or hotkeys intended to conceal assistance.
